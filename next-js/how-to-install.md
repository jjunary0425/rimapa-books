# How to install

## Node JS

[Node JS 페이지](https://nodejs.org/en)에서 LTS를 다운로드한 후 설치

## Next JS

### 어플을 만들 폴더 작성

```
// 폴더 생성
mkdir next-js-example
```

### 프로젝트 생성

```
npx create-next-app@latest
```
